🧹 Tok Clean

Sistema de Gestão - Setup Inicial
1 Verificar Node.js
Certifique-se que tem Node.js instalado:
node --version npm --version
Se não tiver, baixe em: https://nodejs.org/
2 Instalar Dependências
Já foi feito! ✅ (npm install executado)
cd backend npm install
3 Iniciar o Servidor Backend
Execute o comando para iniciar o servidor:
cd backend npm start
Você deve ver: "Servidor rodando em http://localhost:3000"
4 Abrir Frontend no Navegador
Abra o arquivo: frontend/index.html
Clique duplo ou arraste para o navegador
5 Fazer Login
Use as credenciais padrão:
📧 Email:
admin@tokclean.com
🔐 Senha:
admin123
⚠️ Altere a senha após o primeiro login!

npm not found? Instale Node.js: https://nodejs.org/
Porta 3000 em uso? Mude em backend/server.js a constante PORT.
Email já existe? Use outro email ou delete do banco de dados.
